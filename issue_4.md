Title: Support for Slack workspace exports (.json)

Body:
Noticed this while working on the parsers. Right now we only support WhatsApp, Telegram, Instagram, Discord, and Signal. Slack is heavily used for professional connections and "TheAlgorithm" could give amazing insights on workplace communication styles.

Slack workspace exports are just nested folders of JSON files. We could add a parser that reads a ZIP file of the Slack export or just individual JSON files for specific channels. It would be highly valuable for users wanting to analyze professional dynamics.
