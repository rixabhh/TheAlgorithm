How To Export

WhatsApp

  1. Open the chat
  2. Tap menu > More
  3. Export chat > Without Media

Telegram

  1. Open on PC
  2. Tap menu
  3. Export chat history > HTML

Instagram

  1. Go to Your Activity > Download Your Information
  2. Select "Messages" only
  3. Choose JSON format
  4. Extract message_1.json

Discord

  1. Use DiscordChatExporter
  2. Select the DM or channel
  3. Choose JSON format
  4. Save the .json file
