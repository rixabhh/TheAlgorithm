Title: Add shareable "Snapshot Card" for results

Currently, users have no easy way to share their relationship insights with friends or partners. They can only screenshot the entire dashboard (which requires scrolling) or copy raw text, which isn't visually appealing.

We should build a "Snapshot" feature on the dashboard that:
1. Generates a compact, well-designed HTML/canvas element combining the top 3 insights and the visual comparison.
2. Contains only anonymized stats (e.g., using "Person 1" and "Person 2") per our strict privacy rules.
3. Provides a one-click "Download as Image" button to easily share on social media.

This matters because relationship analysis is inherently social. Making the results beautifully shareable acts as built-in growth and significantly improves user satisfaction, rather than forcing them to piece together multiple screenshots.
