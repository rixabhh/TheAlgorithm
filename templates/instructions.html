How to Export Your Chat

Follow these quick steps to export your conversation history. It only takes a minute.

The Algorithm requires your own API Key (BYOK) for 100% privacy and zero middleman servers.

Recommended / Best Free Tier

🤖Google Gemini

  1. Click Here: Go to Google AI Studio API Keys.
  2. Sign in with your Google account.
  3. Click the blue "Create API key" button.
  4. Select an existing project or create a new one.
  5. Copy the key (looks like AIzaSy...) and paste into the settings modal.

Note: Gemini offers a very generous free tier (15 RPM), perfect for heavy chats at $0 cost.

✖️xAI (Grok)

  1. Click Here: Go to xAI Console.
  2. Sign in or create an account.
  3. Navigate to API Keys section.
  4. Click "Create API Key".
  5. Copy the key and paste into the settings modal.

Note: xAI recently introduced a free tier for Grok; check their console for availability.

🧠Anthropic (Claude 3.5)

  1. Click Here: Go to Anthropic Console.
  2. Sign in or create an account.
  3. Navigate to Settings → API Keys.
  4. Click "Create Key". Name it "The Algorithm".
  5. Copy the key (starts with sk-ant-) and paste it in.

Note: Requires $5 minimum billing credits.

🟢OpenAI (GPT-4o)

  1. Click Here: Go to OpenAI API Keys.
  2. Sign in or create a developer account.
  3. Click "Create new secret key".
  4. Copy the key (starts with sk-) immediately.
  5. Paste it into the settings modal.

Note: Requires $5 minimum prepaid credits via Billing.
